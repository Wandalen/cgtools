## Format Guidelines
- **PNG**: Used for raster icons with fixed dimensions
- **SVG**: Used for scalable vector graphics that need to resize without quality loss

## Maintenance
- Maintain consistent dimensions for PNG icons within the same category
- SVG files should use semantic naming and clean paths
- New color swatches should follow the pattern: `[color-name].png`
- New material icons should follow the pattern: `[material-name].png`
- Update this readme when adding or removing icon files
