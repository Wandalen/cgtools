# Character Control

**Keywords:** Animation, Skeletal Animation, glTF, WebGL2, Character Control, Animation Transition

This example demonstrates how to use a renderer crate and its skeletal animation capabilities for creating controllable characters with skeletal animations.

Controls:
- Hold WASD - movement
- Hold Shift + W - running
- Mouse move - camera movement
- Space - standing jump
- Shift + W + Space - running jump
- Hold E - arm kick
- Hold Q - leg kick
